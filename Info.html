Info


Connected Textures

You will need OptiFine for the Connected Textures to work.

To enable Connected Textures go to Options Video Settings Quality Connected Textures = Fancy.


Note: If you are using Clear Glass Borderless turn Connected Textures = OFF.

    Fix Use OptiFine
